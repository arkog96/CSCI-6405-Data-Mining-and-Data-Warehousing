Place the Car_Sales_Data_Set.csv in the same directory as the program file ETL_OLAP.py. 
Run the program ETL_OLAP.py using the command python ETL_OLAP.py 

The program will generate three sorted dataset namely "Car_Sales_Data_Set_First_Sorting.csv", "Car_Sales_Data_Set_Second_Sorting.csv", "Car_Sales_Data_Set_Third_Sorting.csv" in the same directory. 
The porgram then print the 12 tuples lists and prompt the user to enter a input between 1-8.
